# Hari Prasad Professional Portfolio

A modern responsive job-application portfolio for React Native, React.js, Full Stack, Java Spring Boot, and Mobile App Developer roles.

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## EmailJS
Copy `.env.example` to `.env` and add your EmailJS keys.

## Replace
- public/resume.pdf
- src/assets/images/profile.png
- GitHub / LinkedIn / WhatsApp links
- College details
- Live demo links
